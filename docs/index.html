英语语言的学习是英语技能和文化的学习，也是人文素养提高的过程。“高级（人文）英语”课程（Liberal English）立足于外语教学的人文属性，将语言能力的培养与人文素质教育相结合。课程教学突出英语学习的工具性和人文性的统一，既培养学生的英语素养，即促进学生向更高层次的英语学习目标发展；同时注重提高学生综合人文素养，即通过英语课程能够开阔视野，丰富生活经历，形成跨文化意识，发展创新能力，形成良好的品格和正确的人生观和价值观。
